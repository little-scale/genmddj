genmddj WAVE EDITOR

Edit genmddj wavetables (16 × 32 steps, 8-bit, $80=centre; each plays on the YM2612 DAC as a WAVE instrument's shape). Drop a built .bin to edit the factory defaults (the GMDJWAV0 bank every NEW song is seeded with), or a .gmdj to edit that song's 16 waves. Draw, pick a preset, or type an expression. ROM exports re-fix the MD checksum.
