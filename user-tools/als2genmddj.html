als ↔ genmddj

Drop a Live Set (.als) or a Standard MIDI File (.mid) → a genmddj .gmdj song (+ .gmi instruments), or a .gmdj → a .als. The first 9 MIDI tracks map to genmddj channels F1–F6 (FM) then S1–S3 (PSG/TONE) — each FM track's Operator device is converted to a YM2612 patch (also exported as .gmi). Session clips become phrases; a clip > 16 steps spills into more phrases. Contiguous vertical clip runs fill chains; an empty scene starts a new chain. Notes quantise to the 16th grid, octave-fold into range, highest pitch wins on polyphony.
