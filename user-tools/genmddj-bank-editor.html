genmddj BANK EDITOR

Edit the 32-slot SRAM instrument bank — the cross-song library (the middle tier between the factory ROM bank and a song's instruments). Drop a cart or emulator .sav/.srm — linear, EverDrive odd-byte, and gzipped (Genesis Plus) are auto-detected — to edit its bank in place (config + songs untouched), or start a blank cart. Drop .gmi patches to load them into slots. Saves back in the same layout (re-gzipped if the source was).
